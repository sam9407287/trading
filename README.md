# 交易策略圖表集合

交易策略視覺化圖表集合 - 從投資到投機到套利的完整策略流程

## 📊 內容

這個專案包含多個交易策略的 HTML Block Diagram：

1. **交易策略_block_diagram.html** - 完整的策略流程圖，包含：
   - 思維演進階段（投資 → 投機 → 套利）
   - 標的尋找邏輯
   - 策略執行流程
   - 風險與報酬分析
   - 極端風險管制

2. **交易策略流程图.html** - 策略流程圖表

## 🚀 本地運行

```bash
# 安裝依賴
npm install

# 啟動服務器
npm start
```

然後在瀏覽器中訪問 `http://localhost:3000`

## 📦 部署到 Railway

1. 將此專案推送到 GitHub
2. 在 Railway 中創建新專案
3. 連接 GitHub 倉庫
4. Railway 會自動檢測並部署

訪問主頁會看到所有圖表的列表，點擊即可查看。

## 📁 文件說明

- `*.html` - 各種策略視覺化圖表
- `server.js` - Express 服務器（提供文件服務和主頁）
- `package.json` - Node.js 依賴配置
- `railway.json` - Railway 部署配置
