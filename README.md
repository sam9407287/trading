# 交易策略圖表集合

交易策略視覺化圖表集合 - 從投資到投機到套利的完整策略流程

## 📊 內容

這個專案包含多個交易策略的 HTML Block Diagram：

1. **交易策略_思維演進.html** - 思維演進階段獨立頁面，展示：
   - 投資階段 → 投機階段 → 套利階段的完整演進
   - 各階段特點、策略和風險分析
   - 演進邏輯和策略轉換

2. **交易策略_防守型套利.html** - 完整的防守型套利策略流程圖，包含：
   - 標的尋找邏輯
   - 策略執行流程
   - 風險與報酬分析
   - 極端風險管制
   - 完整策略流程圖

3. **交易策略_基礎配置.html** - 基礎配置流程圖表

4. **交易策略_決策樹.html** - 150股模型完整決策樹狀結構，包含：
   - 系統參數設定
   - 階段一：樹幹（初始建倉）
   - 階段二：分岔（週中管理）
   - 階段三：結算（週五收盤）
   - 階段四：枝葉（下週一重啟）
   - 終局：樹的邊界
   - 完整決策樹總覽

## 🚀 本地運行

```bash
# 安裝依賴
npm install

# 啟動服務器
npm start
```

然後在瀏覽器中訪問 `http://localhost:3000`

## 📦 部署到 Railway

1. 將此專案推送到 GitHub
2. 在 Railway 中創建新專案
3. 連接 GitHub 倉庫
4. Railway 會自動檢測並部署

訪問主頁會看到所有圖表的列表，點擊即可查看。

## 📁 文件說明

- `*.html` - 各種策略視覺化圖表
- `server.js` - Express 服務器（提供文件服務和主頁）
- `package.json` - Node.js 依賴配置
- `railway.json` - Railway 部署配置
